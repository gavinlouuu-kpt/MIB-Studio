# MIB-Studio Project Brief

## Project Overview
MIB-Studio is a desktop application for cell analysis in microscopy images. It provides tools for processing, analyzing, and visualizing cell data from both recorded and live microscopy samples.

## Core Requirements
1. Process microscopy images for cell analysis
2. Support both recorded and live image sources
3. Provide visualization and analysis of cell data
4. Maintain high performance for real-time processing

## Project Goals
1. Transition from FTXUI to Qt as the main application framework
2. Continue using OpenCV for all image processing functionality
3. Replace Matplotplusplus with QtCharts for data visualization
4. Maintain or improve current workflow efficiency
5. Create a more modern, intuitive user interface

## Technical Constraints
1. Must maintain compatibility with existing image processing code
2. Must support the same feature set as the current application
3. Must run on Windows platforms (primary target)

## Timeline
- Phase 1: Setup and Initial Integration (current focus)
- Phase 2: Core Component Transition
- Phase 3: Advanced Features and Refactoring
- Phase 4: Full Application Integration
- Phase 5: Testing and Optimization
